## Features

- Automatic key distribution for owned vehicles (on spawn and via `/car` command)
- Simple lock/unlock system (press L)
- Built-in sound effects
- Framework agnostic (via daphne-core)
- Modern notifications (via daphne-notification)
- Optimized performance (0.00ms idle)
- Auto-refresh keys when receiving vehicles

## Requirements

- [daphne-core](https://github.com/daphne-studio/daphne-core)
- [daphne-notification](https://github.com/daphne-studio/daphne-notification)
